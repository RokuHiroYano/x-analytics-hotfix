# X Analytics Heatmap Fixer – Privacy Policy  
X アナリティクスバグの暫定修正 – プライバシーポリシー

Last updated: 2025-11-29  

---

## 1. Overview / 概要

X Analytics Heatmap Fixer (the “Extension”) is a Chrome extension that temporarily fixes the layout bug of the active-hours heatmap on the X (formerly Twitter) analytics “Audience” page.  
X Analytics Heatmap Fixer（以下「本拡張機能」）は、X（旧 Twitter）のアナリティクス「オーディエンス」ページに表示されるアクティブ時間ヒートマップのレイアウト崩れを一時的に修正する Chrome 拡張機能です。

The Extension runs only in the user’s browser and does not send any data to external servers.  
本拡張機能はユーザーのブラウザ内でのみ動作し、外部サーバーにデータを送信しません。

---

## 2. Data collection / データ収集について

The Extension does not collect, store, or transmit any of the following types of data:

- Personally identifiable information (name, email address, etc.)
- Financial or payment information
- Authentication information (passwords, tokens, etc.)
- User communications
- Browsing history
- User activity (clicks, keystrokes, scroll, etc.)
- Website content

本拡張機能は、次のようなデータを収集・保存・送信しません。

- 個人を特定できる情報（氏名、メールアドレスなど）
- 財務情報や支払いに関する情報
- 認証情報（パスワード、トークンなど）
- ユーザー同士のコミュニケーション内容
- ウェブ履歴
- ユーザーのアクティビティ（クリック操作、キーボード入力、スクロールなど）
- 閲覧中のウェブサイトのコンテンツ

The Extension may read the structure of the currently displayed X analytics page only to locate the heatmap element and adjust its layout. This processing happens entirely in the browser’s memory and is not stored or transmitted.  
本拡張機能は、ヒートマップの要素を特定しレイアウトを調整するために、表示中の X アナリティクスページの DOM 構造を読み取りますが、この処理はブラウザ内のメモリ上のみで行われ、保存・送信されることはありません。

---

## 3. Permissions and scope / 権限と適用範囲

The Extension uses the following Chrome permissions:

- `activeTab`
- `scripting`
- Host permissions: `https://x.com/i/account_analytics/audience*`

本拡張機能は、次の Chrome 権限を使用します。

- `activeTab`
- `scripting`
- ホスト権限: `https://x.com/i/account_analytics/audience*`

These permissions are used solely to:

- Inject the layout-fix script into the currently active tab when the user activates the Extension.
- Modify the CSS grid layout of the heatmap element on the X analytics “Audience” page.

これらの権限は、次の目的のためだけに使用されます。

- ユーザーが本拡張機能を有効化したタブに、レイアウト修正用スクリプトを注入すること
- X のアナリティクス「オーディエンス」ページにあるヒートマップ要素の CSS グリッドレイアウトを修正すること

The Extension does not access or modify any other websites or pages.  
本拡張機能がアクセス・変更するのは、上記の X アナリティクスページのみです。
